### Exercício 01

Fazer um programa para ler um número inteiro, e depois dizer se este número é negativo ou não.

Entrada: -10 Saída: NEGATIVO

Entrada: 8 Saída: NAO NEGATIVO

Entrada: 0 Saída: NAO NEGATIVO
